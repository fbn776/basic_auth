# Basic Auth in using MERN stack

This is a sample project to implement basic auth using jwt in MERN stack.

## Tech stack

### Frontend

- React
- Vite
- TypeScript
- Tailwindcss

### Backend

- NodeJS
- Express
- MongoDB

## Usage

### Frontend

```bash
# cd into the frontend code
cd client
# Install all the deps
npm install
# Start the dev server
npm run dev
```
